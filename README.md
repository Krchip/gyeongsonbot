# 경손봇
디스코드 관리봇, 경손봇를 이용해보세요!

## 경손봇은 뭐하는 봇이야?
경손봇은, 디스코드 서버 관리, 검색등 여러가지 기능을 수행할 수 있어요!

`경손아 도움` 을 통해, 경손봇의 기능들을 확인해보세요.

## 경손봇을 초대하고싶어요.
경손봇 초대링크는, [이곳](https://bit.ly/경손봇)을 눌러주세요.

경손봇 [공식 디스코드](https://discord.gg/4uwv3UVEwv) 도 가입해주세요!
